Voici un site dédié à la création d'un manga nommé "Sora no Senso".
Tu peux:
    -voir le scénario dans l'onglet "scénario" (tu peux également poster des idées dans le foorum aux idées)
    -accéder aux nouveautés dans l'onglet dédié
    -voir les post des 3 créateurs du manga
    -nous poser des questions
    -envoyer des messgages à tes amis via un système d'IP

Pour créer un compte, même pas besoin d'adresse e-mail, un nom d'utilisateur et un mot de passe et c réglé
